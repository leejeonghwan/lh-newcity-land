# 신도시 토지 거래 시그니처 (LH-newcity-land)

3기 신도시 발표 전 토지 매집 패턴을 국토부 실거래가 데이터로 정량 분석한 데이터 저널리즘 프로젝트. 2021년 LH 직원 13명이 광명·시흥에서 매집한 시그니처(같은 임야 필지를 다수가 같은 날 지분으로 분할 매수)를 코드로 자동 탐지하고, 다른 신도시 후보지에 같은 코드를 적용해 시그니처가 재현되는지 검증한다.

## 산출물

```
index.html                              # 메인 (지구 비교 카드)
pages/
├── gwangmyeong-siheung.html           # 광명·시흥 상세 분석
└── uiwang-gunpo-ansan.html            # 의왕·군포·안산 + 화성진안 상세 분석
data/
├── raw/                               # 광명·시흥 원본 CSV (국토부)
├── raw_ugas/                          # 의왕·군포·안산·화성 원본 CSV
└── parsed/
    ├── land_all.csv                   # 광명·시흥 통합 (24,765건)
    ├── land_all_ugas.csv              # 의왕·군포·안산 통합 (13,624건)
    ├── viz_data.json                  # 차트용 집계 (광명·시흥)
    └── viz_data_ugas.json             # 차트용 집계 (의왕·군포·안산)
parse.py                                # 광명·시흥 파서
parse_ugas.py                           # 의왕·군포·안산 파서
```

## 주요 결과 (2025-05 기준)

| 지구 | 발표일 | 지구 배율 | 통제군 배율 | 격차 | 발표 후 잔존 |
|------|--------|----------|------------|------|------------|
| 광명·시흥 | 2021-02-24 | ×1.65 | ×1.02 | ×1.62 | 10% |
| 의왕·군포·안산 + 화성진안 | 2021-08-30 | ×1.06 | ×0.65 | ×1.63 | 31% |

- 광명·시흥: 발표 8~12개월 전 지구 7개 동에서 거래 65% 폭증. 같은 임야 필지에 같은 날 17~26명 지분 분할 매수 클러스터 다수.
- 의왕·군포·안산: 격차 배율은 비슷하지만 메커니즘이 다름 — 시장 전반 둔화 속 지구만 거래 유지된 약한 잔존 신호.

## 방법론

1. 각 지구 지정 고시에서 포함된 법정동 확보, 같은 시·구 내 비지정 법정동을 통제군으로 사용.
2. 국토부 실거래가 공개시스템(rt.molit.go.kr)의 사군구별 토지 매매 CSV를 연도별 다운로드.
3. EUC-KR → UTF-8 변환, 데이터 정규화.
4. 시그니처 탐지:
   - 월별 거래 시계열 (지구 vs 통제군)
   - 같은 시군구·동·일자의 5건 이상 클러스터
   - 같은 번지 누적 거래
   - 지분 거래 비율·지목(임야·전·답)
5. 통제군 변화를 baseline으로 사용해 거시 요인을 통제.

## 한계

- 국토부 실거래가는 매수자 신원이 마스킹돼 "누가 샀는가"는 등기부 단계로 넘어가야 한다.
- 본 분석은 "신호가 존재하는가"까지를 데이터로 입증한다.
- 번지는 마지막 자릿수가 `*`로 마스킹돼 동일 필지 여부는 추가 확인 필요.

## 작성·재현 방법

```bash
# 1. 원본 CSV 다운로드 (수동: rt.molit.go.kr에서 시군구·연도별로)
# 2. 파싱
python3 parse.py        # 광명·시흥
python3 parse_ugas.py   # 의왕·군포·안산

# 3. 시각화 데이터 생성 (스크립트 내부 분석 로직)
# 4. index.html 및 pages/*.html을 브라우저에서 열기
```
