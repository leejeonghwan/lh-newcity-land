# 인터넷등기소 — Top 5 클러스터 발급 액션 시트

목적: 데이터로 잡힌 의심 클러스터의 정확한 매수자 분포·지분 분할 구조를 등기부등본 1통(1,000원)으로 확인한다.

## 발급 절차 공통

1. https://www.iros.go.kr 접속, 회원가입 + 본인인증
2. 메뉴: **등기열람/발급 → 부동산 → 간편검색**
3. 입력 구분: **부동산소재지번** 선택
4. 부동산 구분: **토지(또는 산림 — "산" 번지는 임야로 토지로 묶임)**
5. 시·도 → 시·군·구 → 읍·면·동 선택, **본번** 입력 (산 번지는 "산" 체크박스 켜고 번호 입력)
6. 검색 결과 목록에서 정확한 면적·지목 확인 후 발급(1,000원) 또는 열람(700원)
7. PDF 다운로드 → `registry/raw_pdfs/` 폴더에 저장

## 발급 우선순위 — Top 5

각 클러스터의 데이터 신호는 같은 마스킹된 본번에 다수 거래가 누적된 것. 실제 본번은 마지막 자리가 *로 가려져 있어 인접 본번을 모두 시도해야 한다.

### ★★★ 1위 — 하남시 초일동 산1* (1차 3기, 89건)

- **시·도**: 경기도 / **시·군·구**: 하남시 / **읍·면·동**: 초일동
- **검색 본번 후보**: `산10` ~ `산19` (10개 — 산 체크박스 ON)
- **시기 단서**: 2018-01 ~ 2018-12 (1차 3기 발표 12개월 전)
- **거래 평균가**: 약 2,500만원
- **추정**: 단일 필지 89지분 분할 또는 산10~산19 중 2~3개에 누적
- **발급 전략**: `산10`부터 차례로 검색 → 표제부에서 면적·매매 이력 빈도 보고 발급 결정 (5~6통 정도 예상)
- **확인 포인트**: 갑구의 공유자 수·각 지분 비율·매수 일자·매매가

### ★★★ 2위 — 광명시 옥길동 4** (광명·시흥, 54건)

- **시·도**: 경기도 / **시·군·구**: 광명시 / **읍·면·동**: 옥길동
- **검색 본번 후보**: `400` ~ `499` (100개 — 산 체크박스 OFF)
- **시기 단서**: 2020-03 ~ 2020-08 (광명·시흥 발표 6~12개월 전)
- **거래 평균가**: 약 3,600만원
- **클러스터 일자**: 2020-06-23 17건 같은 날 매수 / 2020-06-24 16건 / 2020-07-06 13건
- **추정**: 산-안 붙은 4** 임야 다수 필지 또는 1~2개 핵심 필지
- **발급 전략**: 먼저 광명시청·등기소·지적도(V월드 vworld.kr)로 옥길동 4** 임야 필지 좌표 확인 → 후보 본번 좁히기. 또는 한국토지정보시스템(http://kras.go.kr) "토지대장 열람"으로 본번 검색
- **확인 포인트**: 같은 일자 다수 매수 = 동일 매수 그룹 가능성

### ★★★ 3위 — 하남시 하사창동 산3* (1차 3기, 54건)

- **시·도**: 경기도 / **시·군·구**: 하남시 / **읍·면·동**: 하사창동
- **검색 본번 후보**: `산30` ~ `산39`
- **시기 단서**: 2018-01 ~ 2018-12
- **거래 평균가**: 약 2,100만원

### ★★★ 4위 — 고양시 덕양구 화전동 산6* (2차 3기, 41건)

- **시·도**: 경기도 / **시·군·구**: 고양시 덕양구 / **읍·면·동**: 화전동
- **검색 본번 후보**: `산60` ~ `산69`
- **시기 단서**: 2018-05 ~ 2019-04
- **거래 평균가**: 약 4,700만원

### ★★★ 9위 — 서초구 내곡동 산1* (2024 후보지, 22건)

가장 최근 사례. **2024-11-05 서리풀지구 발표 직전 12개월간 매집**. 강남권이라 매수자 신원 확인 가치 큼.

- **시·도**: 서울특별시 / **시·군·구**: 서초구 / **읍·면·동**: 내곡동
- **검색 본번 후보**: `산10` ~ `산19`
- **시기 단서**: 2023-11 ~ 2024-10
- **거래 평균가**: 약 4,200만원
- **별도 단서**: 같은 내곡동에서 **2024-03-04 대지 클러스터** (15건·80% 지분·평균 26억) — 이건 산 번지가 아닌 일반 번지(3**)이므로 따로 추적

## 본번 좁히기 — 보조 도구

### V월드 (vworld.kr) — 항공사진·지적도

본번이 100개(4**)나 10개(산1*)로 후보가 많을 때:
1. https://map.vworld.kr 접속
2. 시·구·동 검색
3. 우측 도구: "지적도" 토글
4. 필지 클릭 → 본번 확인
5. 가장 큰 임야 필지가 매집 대상일 가능성 큼

### 한국토지정보시스템 KRAS (kras.go.kr) — 토지대장 무료 열람

1. 회원가입 + 본인인증
2. **토지대장 열람**
3. 시·군·구·읍면동 → 본번·부번 입력
4. 무료로 면적·지목·소유자(개인정보 일부 마스킹) 확인 → 발급 대상 좁히기

## 등기부에서 확인할 핵심 정보

각 PDF에서 다음 표 형식으로 옮긴다(`extracted.csv` 스키마 — 본 폴더의 `extracted_template.csv` 참고):

```
필지본번, 면적, 지목, 등기원인, 일자, 매수자성명, 지분비율, 매매금액, 주소
```

특히 주목할 것:
- **공유자 수**: 80명 넘는 필지가 있는지
- **같은 일자 동시 등기**: 2020-06-23, 2020-04-08~10 등 클러스터 일자
- **동일 매수자 다중 필지**: 한 사람이 여러 필지에 분산 매수했는지
- **법인 매수자**: 법인등록번호로 DART·국세청 사업자 조회
- **공직자 의심**: 4급 이상 공직자 재산공개(공보)와 교차

## 다음 단계 (PDF 확보 후)

1. PDF를 `registry/raw_pdfs/{지구}_{지번}.pdf`로 저장
2. `registry/parse_pdf.py` 실행 → 자동 추출
3. 추출된 CSV를 통합 분석 — 동일 매수자 네트워크
4. 강신호 매수자 확인되면 → 공직자 재산공개·국감자료 교차

## 예상 비용·시간

- 발급비: 본번당 1,000원 × 5~10통 = 5,000~10,000원
- 발급 시간: 본번 좁히기 30분 + 발급 자체 10분 = **약 1시간**
- PDF 파싱·정리: 30분 (자동화 완료 후)
- 분석·기사 메모: 2~3시간
