# 등기부 추적 우선순위

광명·시흥·2차 3기·2024 후보지 데이터에서 가장 강한 매집 시그너처를 보이는 필지 30곳. 등기부등본 1통(1,000원)으로 한 필지의 매매 이력·소유자 변동·지분 분할 구조를 확인할 수 있다.

## 추출 기준

- 발표 12개월 전 같은 번지(마스킹된 단위)에 5건 이상 거래가 누적된 경우
- 또는 같은 동·같은 날 5건 이상 묶이고 **지분 비율 ≥ 50%인 클러스터**의 최다 번지
- 지목 우선순위: 임야 > 전·답 > 대 > 도로(자연 매도 가능성 큰 도로는 후순위)

## 발급 절차 (인터넷등기소)

1. https://www.iros.go.kr 접속, 회원가입 + 본인인증
2. 메뉴: 등기열람/발급 → 부동산 → 간편검색
3. 시도 → 시군구 → 읍면동 → 본번/부번 입력 (마스킹된 자리 추정)
4. 등기사항증명서 발급(1,000원) 또는 열람(700원)
5. PDF 다운로드

번지가 `4**` 처럼 마스킹돼 있어 정확한 번지는 일반 지도(다음·네이버 부동산)·국토부 V월드 API로 보강해야 한다. 또는 `4-1`, `4-2` 등 같은 본번의 분할 필지를 차례로 열람.

## 등기부에서 확인할 정보

각 PDF에서 다음을 표로 옮긴다:

- **부동산 표시**: 정확한 지번·면적·지목
- **갑구(소유권에 관한 사항)**: 매매·증여·상속·강제경매 등 권리 변동 이력
  - 시기, 매수자(공유자 다수일 경우 모두), 지분 비율
- **을구(소유권 이외의 권리)**: 근저당·전세권 등
- **표제부**: 토지 분할·합병 이력

## 매수자 식별

등기부는 **공유자 표시**에 매수자 성명과 주소를 보여준다. 단:
- 주민등록번호 뒷자리는 마스킹
- 미성년자·법인은 그대로 표시되는 경우 다수

확인 후 다음 단계:
1. 법인 매수자는 DART·국세청 사업자 조회로 신원 확인
2. 개인 매수자 중 동일 성명·동일 주소가 다수 등장하면 가족·차명 의심
3. 공직자 재산공개(공보 검색)와 교차

## 데이터화

발급된 PDF에서 추출한 정보를 다음 스키마로 정리:

```csv
필지번호,발급일,매수일,매수자성명,매수자주소(부분),지분비율,매매금액,비고
```

`registry/extracted.csv` 파일 생성 권장.
